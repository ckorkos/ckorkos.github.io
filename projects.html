Human-Computer Interaction

Computer Science 170: HCI in Practice



Students worked in groups to needfind, design, test, and code complete web applications. We learned the core aspects of design for easy interaction as well as coding for back end data storage and interactivity. I specialized in the User Experience aspects of the project, but also contributed my knowledge of CSS and JavaScript to the site's functionality.

My partners and I made the site "Simplicipe", which stores recipes for home-cooked meals uploaded by users and ranks them by complexity and time to make. Users could also sort results by number of ingredients, if they are vegan/vegetarian, and whether they contain gluten or not. The site modifies a JSON file to add recipes to the main feed. However, as the course did not have us build a responsive server for the application, these changes are not saved between multiple visits.

Mobile App Design: Rally

Computer Science 199: Rally



This project was completed in a team of three students, myself included. It was done under the guidance of a UCSD professor alongside another team working on their own web app. Each week the teams met to present their progress and receive design critiques.

Rally was an idea that began in a previous class, Cognitive Science 187A: Interaction Design and Information Hierarchy. The idea behind the app's first iteration, RaidIn, was to solve a problem present in many online games that lacked proper matchmaking: the lack of player availability. Our solution was an alert system that could notify a group of players when to meet online. Joining a "Raid" would save these players the often painstaking task of finding strangers to game with.

Game Analysis Paper

Analysis: Star Wars: Knights of the Old Republic I and II

Read the full paper here.

The course Communication 105G: Digital Gaming Studies was centered on computer, console, and mobile gaming. We learned the effects of gaming on both social and personal levels, and the ways in which games are designed to create entertainment, challenge, and purpose. The final paper was an in-depth analysis of a game series.

Star Wars: Knights of the Old Republic is a unique duo of games in that the first plays as an ode to the central themes of Star Wars while the second is a critical interrogation of them. They are not contradictory, but rather different interpretations of the same fiction. This is an artistic quality rarely found in media and as a result it was very enlightening to analyze these stories.

Super Sparty Bros

Unity Project: Super Sparty Bros Platformer

Play the game on your Desktop!

I completed this project as a portion of Coursera's Game Design and Development Specialization. The lessons and assets were provided by Michigan State University, and the first level of the game was a guided tutorial on creatig platformers with concrete goals and challenges. The second level was made entirely by me, using original assets and level design. For the level, I also created an extra life prefab which uses original C# code to increase player lives.

The project taught me more complex level design, hands-on game programming, and several ways in which to create challenge while avoiding frustration.
