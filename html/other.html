Spotify Usability Study

This report details the results of two studies that I conducted on Spotifyâ€™s mobile application. These include a moderated usability test (5 participants) with the goal of discovering how experienced users perform at tasks that gradually increase in complexity, as well as an online survey (42 participants) intended to gather quantitative user data. Color Guessing Game

This was a personal exercise in web development using JavaScript for DOM manipulation and random generation. The goal of the game is relatively simple - match the RGB text displayed above the page to one of the randomized tiles below. The game includes options for Easy/Hard Mode, which changes the number of tiles, and New Game, which resets the board.
